## Description

Rest API capable of CRUD operations built with [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository. and [Postgres](https://github.com/postgres/postgres).

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
